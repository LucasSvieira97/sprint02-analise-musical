# 🎵 Análise de Preferências Musicais — Springfield vs Shelbyville

Projeto desenvolvido durante o Sprint 02 do bootcamp **TripleTen** de Análise de Dados.

## 📋 Objetivo

Testar a hipótese de que a atividade dos usuários de um serviço de streaming é diferente dependendo do dia da semana e da cidade, comparando o comportamento de usuários de Springfield e Shelbyville.

## 🔍 O que foi feito

- Visão geral dos dados e identificação de problemas de qualidade
- Pré-processamento: padronização de cabeçalhos para snake_case
- Tratamento de valores ausentes com substituição por `'unknown'`
- Remoção de duplicatas explícitas
- Análise comparativa de atividade por cidade e dia da semana
- Teste e validação de hipótese com base nos dados

## 🛠️ Ferramentas utilizadas

- Python 3
- Pandas
- Jupyter Notebook
